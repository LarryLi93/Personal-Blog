# 个人博客
一个使用Gemini AI Studio 创建的，基于 Next.js 构建的个人博客，用于展示个人项目、技术文章和个人思考。

# 项目目录

```
- articles/              # 博客文章目录
  - Dive Into LangGraph Plus.md  # LangGraph Plus 深入解析
  - Why you can't change.md      # 为什么你无法改变
- backend/               # 后端服务目录
  - __pycache__/         # Python 编译缓存
  - main.py              # 后端主入口
  - requirements.txt     # Python 依赖
  - start.bat            # Windows 启动脚本
  - start.sh             # Linux/Mac 启动脚本
  - README.md            # 后端说明文档
- components/            # React 组件目录
  - AboutSection.tsx     # 关于我部分组件
  - ArticleCard.tsx      # 文章卡片组件
  - ArticleGrid.tsx      # 文章网格布局组件
  - ArticleView.tsx      # 文章详情页组件
  - Footer.tsx           # 页脚组件
  - Header.tsx           # 页眉组件
  - Hero.tsx             # 英雄区组件
  - TableOfContents.tsx  # 文章目录组件
- imgs/                  # 图片资源目录
  - Intellectual Obesity.png  # 智力肥胖文章封面
  - author.ico           # 作者图标
  - author.png           # 作者头像
  - dive-into-langgraph-plus.jpg  # LangGraph Plus 文章封面
- services/              # 服务层目录
  - articleService.ts    # 文章相关服务
  - globalVisitService.ts  # 全局访问统计服务
  - visitService.ts      # 访问统计服务
- utils/                 # 工具函数目录
  - userUtils.ts         # 用户相关工具函数
- .gitignore             # Git 忽略配置
- App.tsx               # React 应用主组件
- README.md             # 项目说明文档
- index.html            # HTML 入口文件
- index.tsx             # React 应用入口
- metadata.json         # 元数据配置
- package-lock.json     # NPM 依赖锁定文件
- package.json          # NPM 项目配置
- tsconfig.json         # TypeScript 配置
- types.ts              # TypeScript 类型定义
- vite-env.d.ts         # Vite 环境类型声明
- vite.config.ts        # Vite 配置
```
